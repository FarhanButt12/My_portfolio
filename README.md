# 🚀 Muhammad Farhan Mukhtar Butt — Developer Portfolio

A premium, responsive, and interactive personal portfolio website showcasing mobile application development, full-stack web solutions, and machine learning projects.

Built with **React.js**, **Vite**, **Tailwind CSS**, and **Framer Motion**.

---

## 🌟 Website Sections & Architecture

The project consists of a single-page application split into highly detailed modular sections:

- **Home**: Hero section featuring a developer badge, social media links, and a dynamic typewriter introduction (`react-type-animation`).
- **About Me**: Narrative of academic background, timeline, and core soft skill tags.
- **Education**: Detailed foundation timeline showcasing university degree studies, coursework, and highlights.
- **What I'm Looking For / Development Focus**: Overview of development focus, core methodology (Agile, unit tests, code reviews), and collaboration preferences.
- **Technical Skills**: A grouped grid of technical skills with hover states:
  - **Languages**: Dart, Python, JavaScript, C++, HTML5, CSS3
  - **Frameworks & Tools**: Flutter, Firebase, MongoDB, Android Studio, VS Code, Git
  - **ML Algorithms**: KNN, Naïve Bayes, SVM, Neural Networks
  - **Areas of Interest**: App Development, Web Dev, Machine Learning
- **Certifications**: List of credential achievements (complete web bootcamps, Google AI courses).
- **Projects**: Showcase of key projects with interactive sheets:
  1. **AI-Based Cyber Attack Detection System** (Python, Scikit-learn)
  2. **Cross-Platform Expense Tracker App** (Flutter, Firebase)
  3. **UOG Official Website Redesign** (HTML, CSS, JS)
- **Activities & Leadership**: Summary of university society engagements, coding competition hacks, and team leadership.
- **Availability Banner**: Interactive glowing status card advertising freelance and job availability.
- **Contact Me**: Direct feedback email contact form connected to FormSubmit and interactive social channels.
- **Resume Print View**: A dedicated CSS media-query print view optimized for printing a clean, standard CV direct from the browser (`Ctrl + P`).

---

## ⚡ Key Features

- **Modern & Glowing Aesthetics**: Dark-mode design accented with vibrant gradients (cyan to violet) and glowing background grid layout.
- **Responsive Layout**: Designed to look premium across Desktop, Tablet, and Mobile devices.
- **Micro-Animations**: Framer Motion scroll triggers, card hover scaling, and active button click responses.
- **Resume Download**: Styled header CTA button with a seamless file download for the latest resume PDF.
- **Print Optimization**: Hides normal website containers in favor of a specialized clean CV template when the browser prints (`Ctrl + P`).
